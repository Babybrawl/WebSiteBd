Étape 1 : Ouvrir Firefox sur votre appareil Android

- Assurez-vous que Firefox est installé et ouvrez-le sur votre appareil.

Étape 2 : Accéder au site d'extension uBlock Origin

- Dans la barre d'adresse de Firefox, tapez addons.mozilla.org. - Recherchez "uBlock Origin" dans la barre de recherche sur le site addons.mozilla.org.

Étape 3 : Sélectionner uBlock Origin

- Une fois que vous avez trouvé uBlock Origin dans les résultats de recherche, appuyez dessus pour ouvrir la page de l'extension.

Étape 4 : Installer uBlock Origin

- Sur la page de l'extension uBlock Origin, vous verrez un bouton "Ajouter à Firefox". - Appuyez sur ce bouton pour démarrer le processus d'installation.

Étape 5 : Confirmer l'installation

- Firefox vous demandera de confirmer que vous souhaitez ajouter l'extension. Appuyez sur "Ajouter" pour continuer.

Étape 6 : Terminer l'installation

- L'extension uBlock Origin sera téléchargée et installée

Étape 1 : Ouvrir Google Chrome

- Lancez votre navigateur Google Chrome.

Étape 2 : Accéder au Chrome Web Store

- Dans la barre d'adresse, tapez `https://chrome.google.com/webstore` et appuyez sur Entrée.

Étape 3 : Rechercher uBlock Origin

- Dans la barre de recherche située en haut à gauche de la page, tapez "uBlock Origin" et appuyez sur Entrée.

Étape 4 : Sélectionner uBlock Origin

- Dans les résultats de recherche, trouvez l'extension "uBlock Origin" développée par Raymond Hill (gorhill). - Cliquez sur "uBlock Origin" pour accéder à la page de l'extension.

Étape 5 : Ajouter à Chrome

- Sur la page de l'extension, cliquez sur le bouton bleu "Ajouter à Chrome".

Étape 6 : Confirmer l'installation

- Une fenêtre contextuelle apparaîtra pour confirmer l'ajout de l'extension. Cliquez sur "Ajouter l'extension".

Étape 7 : Vérifier l'installation

- Une fois l'installation terminée, vous verrez une icône de bouclier rouge avec un "u" blanc apparaître dans la barre d'outils de Chrome, indiquant que l'extension uBlock Origin a été correctement installée.

Étape 8 : Configurer uBlock Origin (optionnel)

- Cliquez sur l'icône de l'extension pour ouvrir uBlock Origin. - Vous pouvez accéder aux paramètres en cliquant sur l'icône d'engrenage. Ici, vous pouvez ajuster les filtres et autres préférences selon vos besoins. Et voilà ! Vous avez maintenant installé l'extension uBlock Origin sur Google Chrome pour améliorer votre expérience de navigation en bloquant les publicités et autres contenus indésirables. Pour plus de détails ou de support, vous pouvez visiter la page d'aide officielle de uBlock Origin sur le [Chrome Web Store](https://chrome.google.com/webstore).
